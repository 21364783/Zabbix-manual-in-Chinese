1 获得Zabbix
=================================

.. contents::


概述
---------------------------

有三种方法可以获得Zabbix:

   * 通过 `发布版软件包` 安装它
   * 下载最新的源码压缩包并 `自己编译它`
   * 下载 `虚拟应用`

下载源码或许虚拟应用，请访问 `Zabbix下载页面 <http://www.zabbix.com/download.php>`_ , 该页提供有最新版本。 如果想下载过去的版本，请点击 stable版本下载下边的连接
